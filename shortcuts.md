
# Useful shortcuts (WIP)

## Mac OS

### go to end of line:
⌘ + →
### go to beginning of line:
⌘ + ←
### go to prev word:
alt + ← 
### go to next word:
alt + →
### spotlight search
⌘ + space


## Chrome

### Go to address bar
⌘ + L
### Switch tab
⌘ + alt + →

⌘ + alt + ←
### New Tab
⌘ + T
### Close tab
⌘ + W
### Quit (close all tabs)
⌘ + Q

## iTerm

### New Tab
⌘ + T
### Close tab
⌘ + W
### Quit (close all tabs)
⌘ + Q

## Slack

### Move up/down conversation
alt + ↑

alt + ↓


## Lastpass

### Bring up password quicksearch
⇧ + ⌘ + L
