
# Useful shortcuts (WIP)

## Mac OS

### go to end of line:
⌘ + →
### go to beginning of line:
⌘ + ←
### go to prev word:
alt + ← 
### go to next word:
alt + →
### spotlight search
⌘ + space
### mouse gestures
https://support.apple.com/en-gb/HT204895

## Chrome

### Go to address bar
⌘ + L
### Switch tab
⌘ + alt + →

⌘ + alt + ←
### New Tab
⌘ + T
### Close tab
⌘ + W
### Quit (close all tabs)
⌘ + Q
